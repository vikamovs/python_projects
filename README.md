# Mad Libs Story Generator

Create your own funny stories with this Python program! You choose a story template and enter words like nouns, verbs, adjectives, colors, and more. The program will use your words to generate a silly, personalized story.  

## How to Use

1. Run project1.py.  
2. Choose a template (1, 2, or 3).  
3. Enter words when prompted.  
4. Enjoy your custom story printed on the screen!  

## Requirements

- Python 3.x
